# Logout Instance - Evolution API Documentation

## Introdução
Endpoint para realizar o logout de uma instância na Evolution API.

### Método
**DELETE**

### Endpoint
`/instance/logout/{instance}`

### Exemplo de cURL
```bash
curl --request DELETE \
  --url https://{server-url}/instance/logout/{instance} \
  --header 'apikey: <api-key>'
Respostas
Sucesso (200)
Código: 200

JSON

{
  "status": "SUCCESS",
  "error": false,
  "response": {
    "message": "Instance logged out"
  }
}
Erro (404)
Código: 404

Autorização
apikey (string) - requerido: Sua chave de autorização no cabeçalho.
Parâmetros de Caminho
instance (string) - requerido: Nome da instância a ser desconectada.
Tipo de Resposta
200 - application/json

A resposta é do tipo object.
