# Send Presence - Evolution API Documentation

## Introdução
Endpoint para enviar a presença de um usuário em um chat usando a Evolution API.

### Método
**POST**

### Endpoint
`/chat/sendPresence/{instance}`

### Exemplo de cURL
```bash
curl --request POST \
  --url https://{server-url}/chat/sendPresence/{instance} \
  --header 'Content-Type: application/json' \
  --header 'apikey: <api-key>' \
  --data '{
    "number": "<string>",
    "options": {
      "delay": 123,
      "presence": "composing",
      "number": "<string>"
    }
  }'
Autorização
apikey (string) - requerido: Sua chave de autorização no cabeçalho.
Parâmetros de Caminho
instance (string) - requerido: Nome da instância de chat.
Tipo de Body
application/json
Resposta
A resposta será uma confirmação do envio da presença, embora a estrutura exata do corpo de resposta não esteja especificada.
