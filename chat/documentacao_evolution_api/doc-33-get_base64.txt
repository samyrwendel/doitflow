# Get Base64 - Evolution API Documentation

## Introdução
Endpoint para obter a representação em base64 de uma mensagem de mídia usando a Evolution API.

### Método
**POST**

### Endpoint
`/chat/getBase64FromMediaMessage/{instance}`

### Exemplo de cURL
```bash
curl --request POST \
  --url https://{server-url}/chat/getBase64FromMediaMessage/{instance} \
  --header 'Content-Type: application/json' \
  --header 'apikey: <api-key>' \
  --data '{
    "message": {
      "key": {
        "id": "<string>"
      }
    },
    "convertToMp4": true
  }'
Autorização
apikey (string) - requerido: Sua chave de autorização no cabeçalho.
Parâmetros de Caminho
instance (string) - requerido: Nome da instância de chat.
Tipo de Body
application/json
Resposta
Sucesso (200)

JSON

{
  "base64": "<string>"
}
Descrição da Resposta
base64: A representação em base64 da mídia solicitada.
