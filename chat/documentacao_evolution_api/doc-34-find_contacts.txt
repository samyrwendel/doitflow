## Introdução
Endpoint para buscar informações de contatos em um chat usando a Evolution API.

### Método
**POST**

### Endpoint
`/chat/findContacts/{instance}`

### Exemplo de cURL
```bash
curl --request POST \
  --url https://{server-url}/chat/findContacts/{instance} \
  --header 'Content-Type: application/json' \
  --header 'apikey: <api-key>' \
  --data '{
    "where": {
      "id": "<string>"
    }
  }'
Autorização
apikey (string) - requerido: Sua chave de autorização no cabeçalho.
Parâmetros de Caminho
instance (string) - requerido: Nome da instância de chat.
Tipo de Body
application/json
Resposta
A resposta será em formato JSON, contendo informações sobre os contatos encontrados.

JSON

{
  "contacts": [
    {
      "id": "123456789@s.whatsapp.net",
      "name": "John Doe",
      "profilePictureUrl": "https://example.com/profile.jpg"
    }
  ]
}
Descrição da Resposta
contacts: Uma lista de objetos contendo informações dos contatos, incluindo id, name e profilePictureUrl.
