## Introdução
Endpoint para arquivar uma conversa específica usando a Evolution API.

### Método
**POST**

### Endpoint
`/chat/archiveChat/{instance}`

### Exemplo de cURL
```bash
curl --request POST \
  --url https://{server-url}/chat/archiveChat/{instance} \
  --header 'Content-Type: application/json' \
  --header 'apikey: <api-key>' \
  --data '{
    "lastMessage": {
      "key": {
        "remoteJid": "<string>",
        "fromMe": true,
        "id": "<string>"
      }
    },
    "archive": true,
    "chat": "<string>"
  }'
Respostas
Sucesso (201)
Código: 201

JSON

{
  "chatId": "553198296801@s.whatsapp.net",
  "archived": true
}
Autorização
apikey (string) - requerido: Sua chave de autorização no cabeçalho.
Parâmetros de Caminho
instance (string) - requerido: Nome da instância para arquivar a conversa.
Tipo de Body
application/json
Tipo de Resposta
201 - application/json
A resposta é do tipo object.
