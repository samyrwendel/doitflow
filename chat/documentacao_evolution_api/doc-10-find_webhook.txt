## Introdução
Endpoint para buscar os detalhes do webhook de uma instância existente na Evolution API.

### Método
**GET**

### Endpoint
`/webhook/find/{instance}`

### Exemplo de cURL
```bash
curl --request GET \
  --url https://{server-url}/webhook/find/{instance} \
  --header 'apikey: <api-key>'
Respostas
Sucesso (200)
Código: 200

JSON

{
  "enabled": true,
  "url": "https://example.com",
  "events": [
    "APPLICATION_STARTUP"
  ]
}
Erro (404)
Código: 404

Autorização
apikey (string) - requerido: Sua chave de autorização no cabeçalho.
Parâmetros de Caminho
instance (string) - requerido: Nome da instância para buscar o webhook.
Tipo de Resposta
200 - application/json

A resposta é do tipo object.
