## Introdução
Endpoint para atualizar o status de bloqueio de um usuário em um chat usando a Evolution API.

### Método
**POST**

### Endpoint
`/message/updateBlockStatus/{instance}`

### Exemplo de cURL
```bash
curl --request POST \
  --url https://{server-url}/message/updateBlockStatus/{instance} \
  --header 'Content-Type: application/json' \
  --header 'apikey: <api-key>' \
  --data '{
    "number": "<string>",
    "status": "<string>"
  }'
Autorização
apikey (string) - requerido: Sua chave de autorização no cabeçalho.
Parâmetros de Caminho
instance (string) - requerido: Nome da instância de chat.
Tipo de Body
application/json
Resposta
A resposta será uma confirmação da atualização do status de bloqueio, embora a estrutura exata do corpo de resposta não esteja especificada.
