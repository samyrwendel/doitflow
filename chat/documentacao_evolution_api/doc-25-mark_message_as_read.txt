# Mark Message As Read - Evolution API Documentation

## Introdução
Endpoint para marcar mensagens como lidas em uma conversa específica usando a Evolution API.

### Método
**POST**

### Endpoint
`/chat/markMessageAsRead/{instance}`

### Exemplo de cURL
```bash
curl --request POST \
  --url https://{server-url}/chat/markMessageAsRead/{instance} \
  --header 'Content-Type: application/json' \
  --header 'apikey: <api-key>' \
  --data '{
    "readMessages": [
      {
        "remoteJid": "<string>",
        "fromMe": true,
        "id": "<string>"
      }
    ]
  }'
Respostas
Sucesso (201)
Código: 201

JSON

{
  "message": "Read messages",
  "read": "success"
}
Autorização
apikey (string) - requerido: Sua chave de autorização no cabeçalho.
Parâmetros de Caminho
instance (string) - requerido: Nome da instância para marcar as mensagens como lidas.
Tipo de Body
application/json
Tipo de Resposta
201 - application/json
A resposta é do tipo object.
