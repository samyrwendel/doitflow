## Introdução
Endpoint para deletar uma mensagem específica para todos os participantes de um chat usando a Evolution API.

### Método
**DELETE**

### Endpoint
`/chat/deleteMessageForEveryone/{instance}`

### Exemplo de cURL
```bash
curl --request DELETE \
  --url https://{server-url}/chat/deleteMessageForEveryone/{instance} \
  --header 'Content-Type: application/json' \
  --header 'apikey: <api-key>' \
  --data '{
    "id": "<string>",
    "remoteJid": "<string>",
    "fromMe": true,
    "participant": "<string>"
  }'
Resposta
Sucesso (201)

Autorização
apikey (string) - requerido: Sua chave de autorização no cabeçalho.
Parâmetros de Caminho
instance (string) - requerido: Nome da instância de chat.
Tipo de Body
application/json
Tipo de Resposta
201 - application/json: A resposta é do tipo objeto representando uma mensagem de protocolo do WhatsApp, incluindo a chave, conteúdo da mensagem, timestamp e status.
