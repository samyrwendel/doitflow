## Introdução
Endpoint para enviar um status para um número específico usando a Evolution API.

### Método
**POST**

### Endpoint
`/message/sendStatus/{instance}`

### Exemplo de cURL
```bash
curl --request POST \
  --url https://{server-url}/message/sendStatus/{instance} \
  --header 'Content-Type: application/json' \
  --header 'apikey: <api-key>' \
  --data '{
    "type": "text",
    "content": "<string>",
    "caption": "<string>",
    "backgroundColor": "<string>",
    "font": 123,
    "allContacts": true,
    "statusJidList": [
      "{{remoteJID}}"
    ]
  }'
