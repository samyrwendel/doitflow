## Introdução
Endpoint para enviar informações de contato de uma pessoa para um número específico usando a Evolution API.

### Método
**POST**

### Endpoint
`/message/sendContact/{instance}`

### Exemplo de cURL
```bash
curl --request POST \
  --url https://{server-url}/message/sendContact/{instance} \
  --header 'Content-Type: application/json' \
  --header 'apikey: <api-key>' \
  --data '{
    "number": "<string>",
    "contact": [
      {
        "fullName": "<string>",
        "wuid": "<string>",
        "phoneNumber": "<string>",
        "organization": "<string>",
        "email": "<string>",
        "url": "<string>"
      }
    ]
  }'
